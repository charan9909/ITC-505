Lights Out

Click tiles to toggle them and their neighbors. Turn all lights off to win!

Development Addendum

Creative Process: I designed a clean grid with color contrast to represent the “on” and “off” states. I wanted the interface to be intuitive and fast to interact with.

Challenges: Randomizing the game while ensuring it's always solvable was a challenge. I overcame this by simulating a set of random user clicks on a blank board. This guarantees a solvable configuration each time.

Insights: This project taught me the importance of separating concerns in web development — styling, logic, and structure. I also learned how to manipulate the DOM effectively and detect win conditions based on CSS class states.

Font Handling: Default fonts were used to ensure consistent cross-platform rendering. No custom font serving was needed in this version.
